【 概要 】
スプレッドシートで、スケジュール進捗がわかる工事台帳

【 依頼内容 】
元となる工事台帳のデータはあります。

①物件シートを新たに作成すると進捗中案件シートに反映される
②物件シートに着工日・引き渡し日を記入（月単位でok）
③進捗スケジュールシートにスケジュール感がわかるヨコ棒グラフができる
④進捗スケジュールシートに完工チェックマークセルなどがあり、それを✓するとその物件シートは、進捗中案件シートから自動的に完工済案件シートに移行する
